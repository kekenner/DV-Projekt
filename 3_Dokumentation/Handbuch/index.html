Einleitung

Spielregeln

Das Spiel wird auf einem senkrecht stehenden hohlen Spielbrett gespielt,

in das die Spieler abwechselnd ihre Spielsteine fallen lassen.

Das Spielbrett besteht aus senkrechten Spalten und waagerechten Reihen.

Jeder Spieler besitzt eigenfarbige Spielsteine. Wenn ein Spieler einen Spielstein in eine Spalte fallen lässt,

besetzt dieser den untersten freien Platz der Spalte. Gewinner ist der Spieler,

der es als erster schafft, vier oder mehr seiner Spielsteine waagerecht, senkrecht oder diagonal in eine Linie zu bringen.

Das Spiel endet unentschieden, wenn das Spielbrett komplett gefüllt ist, ohne dass ein Spieler eine Viererlinie gebildet hat.

Programmnutzung

Nachdem Starten des Programms erscheint folgendes Fenster:

Wenn Sie der erste Spieler sind der das Programm gestartet hat, wählen Sie den Button „Server starten“.

Anschließend erhalten Sie eine Nachricht „Warten auf Gegenspieler…“, die Sie bestätigen müssen.

Wenn von einem anderen Spieler bereits ein Server gestartet wurde, können Sie den Button „Mit Server verbinden…“ betätigen.

Im Zuge dessen erscheint ein neues Fenster indem Sie die IP-Adresse des Gegners eintragen.

Nachdem die IP-Adresse erfolgreich eingegeben wurde erscheint beim ersten Spieler eine Nachricht „Gegenspieler Gefunden!“,

die ebenfalls bestätigt werden muss.

Anschließend erscheint bei beiden Spielern ein neues Fenster in dem Sie ihr Name Eintragen können.

Nachdem beide Spieler ihren Namen eingetragen haben kann gespielt werden.

Je nachdem welcher Spieler am Zug ist, färbt sich das Spielfeld blau, ist der andere Spieler am Zug grau.

Durch klicken in die gewählte Spalte setz man sein Zug.

IP-Adresse finden:

Die IP-Adresse findet man unter Netzwerk -& Interneteinstellungen bei Eigenschaften.
